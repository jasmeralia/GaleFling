# Roadmap

## Phase 0 (Previous)
- Twitter + Bluesky posting
- Single optional image
- Core UX: platform selection, preview, error handling, logs, updates

## Phase 1 (Current)
- Additional platforms (Instagram, Fetlife, OnlyFans,  Fansly, etc.)
- Multiple images per post
- Platform-specific text (per-platform captions)
- Hashtag management
- Optional scheduled posting

## Phase 2
- Video support (FFmpeg encoding, progress, chunked uploads)
- Platform-specific video constraints
- macOS build pipeline
